配置主题

这里我选择使用next作为主题 gitHub地址
1.下载代码

  • 方式一：通过git下载

    cd [站点文件夹]
git clone https://github.com/theme-next/hexo-theme-next themes/next

  • 方式二：将github中代码复制到themes文件夹下面即可

2.修改根目录下_config.yml配置文件即可

设置分类

1.创建“分类”选项页

2.打开source/categories/index.md,添加type: “categories”到内容中

3.给文章添加“categories”属性

打开之前存在的文章添加categories: 属性即可完成分类功能

添加Tag

1.创建“标签”选项页

2.打开source/tags/index.md，添加type: “tags”到内容中，添加后如下

3.给文章添加“tags”属性
打开需要添加标签的文章，为其添加tags属性。下方的tags:下方的- jQuery - 表格
- 表单验证就是这篇文章的标签了

开启浏览量访问

(这里只针对next主题而言，其他主题可能需要其他的一些配置)
在themes/_config.yml中开启busuanzi统计功能即可

开通评论功能

使用Valine 评论系统：这款系统不需要登录便可以登录，没有后端，Next 主题最新已经支持安装
建议使用最新版next

1.在themes/_config.yml的配置文件中开启valine功能

其中的appid和appkey需要去官网创建应用获得，地址如下：

LeanCloud官网登录入口

创建应用，选择开发版，创建好之后就能获取到id和key复制到上一步即可

2.关闭评论功能
对于不想开启评论的文章，只需要添加comments字符设置为false即可
